
# HQChart C++
麦语法指标计算引擎c++库。兼容通达信，东方财富，文华等脚本指标.


# hqchartPy2
hqchartPy2使用cpython开发， 把麦语法指标计算引擎cc++库分装下给py调用。支持批量指标计算。可以用于后台K线指标计算，后台选股。<br>
**支持平台** windows, linux.  不同平台需要自行编译. 只支持64位平台(py也需要时64位的)<br>

源码目录:[HQChartPy2.Free](/HQChartPy2.Free)
